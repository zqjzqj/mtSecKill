## 主要功能
- 抢购飞天茅台  
- 登陆京东商城（[www.jd.com](http://www.jd.com/)）
  - 使用进程调起的浏览器登陆京东 【无需其他任何操作】 
- 秒杀预约后等待抢购
  - 定时开始自动抢购
  - 抢购过程中浏览器会打开对应works数的窗口，请不要大惊小怪 也不要关闭
